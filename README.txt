Usage:

$ . port.sh "port.zip" "base.zip"
base.zip and port.zip are default name of this args

base.zip = ".zip file of base ROM"
port.zip = ".zip file of ROM to port"

eg. with base.zip as base
$ . port.sh a1174528.zip
